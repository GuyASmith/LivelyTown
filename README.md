## For Translators

You can create a pull request with a given translation file addition or change. Translations are difficult to verify, however, so it may take time to approve.

## For Modders

If you have a code fix for an existing issue or mod compatibility, create a pull request with the change. Please be sure to explain your logic in the pull request or code comments. If I like your changes, but require more comments or have change suggestions, it will be up to you to implement them.
